# 504. Base 7

______________________________________________________________________

## 📄 Problem Summary

Diberikan integer `num`, kembalikan representasinya dalam **basis 7** sebagai string.

Contoh:

- `num = 100` → `"202"` (2×49 + 0×7 + 2×1 = 100)
- `num = -7` → `"-10"` (-1×7 + 0 = -7)

______________________________________________________________________

## 💡 Intuition

Konversi basis menggunakan **repeated division** — ambil sisa bagi 7 (`num % 7`) sebagai digit dari kanan ke kiri, bagi `num` dengan 7, ulangi sampai habis. Untuk bilangan negatif, proses nilai absolutnya lalu tambahkan tanda `-` di akhir.

```
100 → basis 7:
100 % 7 = 2 (digit paling kanan)
100 / 7 = 14
14 % 7 = 0
14 / 7 = 2
2 % 7 = 2 (digit paling kiri)
2 / 7 = 0 → stop

Digits dari kanan: 2, 0, 2 → "202" ✅
```

______________________________________________________________________

## 🔍 Approach

### Repeated Division by 7

1. Jika `num == 0` → return `"0"` langsung.
1. Simpan tanda negatif, ambil `num = Math.abs(num)`.
1. Loop selama `num != 0`:
   - Append `num % 7` ke StringBuilder.
   - `num /= 7`.
1. Jika negatif → append `"-"`.
1. Return `sb.reverse().toString()`.

| | |
| --------- | -------------------------------------- |
| **Time** | O(log₇ n) — jumlah digit dalam basis 7 |
| **Space** | O(log₇ n) — StringBuilder untuk hasil |

## 🧪 Dry Run

**Input:** `num = 100`

`isNegative=false, num=100`

| num | num%7 | append | sb | num/7 |
| --- | ----- | ------ | ----- | ----- |
| 100 | 2 | '2' | "2" | 14 |
| 14 | 0 | '0' | "20" | 2 |
| 2 | 2 | '2' | "202" | 0 |

`num=0` → stop. `sb.reverse()` = `"202"`

**Output: `"202"` ✅**

______________________________________________________________________

**Input:** `num = -7`

`isNegative=true, num=7`

| num | num%7 | append | sb | num/7 |
| --- | ----- | ------ | ---- | ----- |
| 7 | 0 | '0' | "0" | 1 |
| 1 | 1 | '1' | "01" | 0 |

Append `"-"` → `"01-"`. `sb.reverse()` = `"-10"`

**Output: `"-10"` ✅**

______________________________________________________________________

**Input:** `num = 0`

Return `"0"` langsung (early return).

**Output: `"0"` ✅**

## ⚠️ Edge Cases

- [ ] `num = 0` → return `"0"` (tanpa early return, loop tidak jalan dan sb kosong)
- [ ] `num` negatif → simpan tanda, proses nilai absolut
- [ ] `num` = pangkat 7 (7, 49, 343) → representasi berupa `"10"`, `"100"`, `"1000"`

______________________________________________________________________

## 🔧 Kenapa Early Return untuk `num == 0`?

```java
if (num == 0) return "0";
```

Tanpa ini, `while (num != 0)` tidak jalan sama sekali → `sb` kosong → `sb.reverse().toString() = ""` — return string kosong, bukan `"0"`!

______________________________________________________________________

## 🔧 Kenapa Append `"-"` Setelah Loop, Bukan Sebelum?

```java
// Kode ini benar:
while (num != 0) { sb.append(num % 7); num /= 7; }
if (isNegative) sb.append("-");
return sb.reverse().toString();
```

Kita append digit dari kanan ke kiri ke `sb`, lalu reverse. Jika `"-"` diappend terakhir, setelah reverse ia akan ada di posisi pertama (kiri) — persis yang kita inginkan.

```
sb sebelum reverse: "01-"
sb setelah reverse: "-10" ✅
```

## 📌 Key Takeaway

Konversi basis adalah aplikasi klasik dari **repeated division** — sisa bagi (`% base`) memberikan digit dari kanan ke kiri, pembagian (`/ base`) memindah ke digit berikutnya. Dua detail penting: early return untuk `0` agar tidak return string kosong, dan tanda negatif diproses terpisah dari angkanya. 🎯
